# GuardPup Landing Page

A futuristic landing page for GuardPup - the cyber smart dog security camera.

## 🚀 Features

- Custom bone-shaped cursor with paw print trails
- Floating bone and paw treats
- Purple/aqua futuristic color scheme
- Smooth animations and glowing effects
- Fully responsive design
- No dependencies - standalone HTML file

## 📦 Deployment to GitHub Pages

This is a single-file HTML website ready for GitHub Pages deployment.

### Quick Deploy

1. Upload `index.html` to your GitHub repository
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/ (root)` folder
5. Save and wait 1-2 minutes

Your site will be live at: `https://YOUR-USERNAME.github.io/REPO-NAME/`
